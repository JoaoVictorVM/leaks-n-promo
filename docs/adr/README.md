# Architecture Decision Records (ADRs)

Este diretório registra as **decisões de arquitetura** do projeto no formato
[MADR](https://adr.github.io/madr/). Cada ADR é imutável: quando uma decisão
muda, cria-se um novo ADR que **substitui** o anterior (em vez de editá-lo).

- Use o [template](template.md) como ponto de partida para um novo ADR.
- Numere sequencialmente (`NNNN-titulo-curto.md`).
