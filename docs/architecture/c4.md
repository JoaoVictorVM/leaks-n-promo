# Diagramas C4 — leaks&promo

Modelo [C4](https://c4model.com/) em Mermaid (renderiza nativamente no GitHub),
nos níveis **Contexto → Container → Componente**. Os diagramas refletem a
arquitetura pull-only e stateless da v1 (ver [ADRs](../adr/README.md)).

## Nível 1 — Contexto

```mermaid
C4Context
    title Contexto do sistema - leaks&promo
    Person(user, "Usuário do Discord", "Membro de um servidor interessado em games")
    System(bot, "leaks&promo", "Bot de Discord que responde /preco e /leaks")
    System_Ext(discord, "Discord", "Plataforma e gateway de mensagens")
    System_Ext(cheapshark, "CheapShark API", "Preços de jogos de PC (USD)")
    System_Ext(rss, "Feeds RSS", "Sites de notícias/leaks de games")
    System_Ext(reddit, "Reddit API", "r/GamingLeaksAndRumours (opcional)")

    Rel(user, discord, "Usa comandos via")
    Rel(discord, bot, "Encaminha interações (gateway)")
    Rel(bot, cheapshark, "Consulta preços", "HTTPS")
    Rel(bot, rss, "Busca leaks", "HTTPS")
    Rel(bot, reddit, "Busca leaks (se aprovado)", "HTTPS/OAuth")
```

## Nível 2 — Container

```mermaid
C4Container
    title Containers do sistema - leaks&promo
    Person(user, "Usuário do Discord")
    System_Ext(discord, "Discord Gateway")
    System_Ext(cheapshark, "CheapShark API")
    System_Ext(rss, "Feeds RSS")
    System_Ext(reddit, "Reddit API")

    Container_Boundary(bot, "leaks&promo (processo Go)") {
        Container(gateway, "Camada Discord", "discordgo", "Gateway, registro e handlers de comandos")
        Container(price, "Módulo de Preço", "Go", "PriceProvider + client CheapShark")
        Container(leaks, "Módulo de Leaks", "Go", "LeakSource: RSS + Reddit")
        ContainerDb(cache, "Cache", "in-memory, TTL", "Memoriza respostas recentes")
    }

    Rel(user, discord, "Comandos /preco /leaks")
    Rel(discord, gateway, "Interações", "websocket")
    Rel(gateway, price, "Consulta")
    Rel(gateway, leaks, "Consulta")
    Rel(price, cache, "lê/escreve")
    Rel(leaks, cache, "lê/escreve")
    Rel(price, cheapshark, "Consulta", "HTTPS")
    Rel(leaks, rss, "Busca", "HTTPS")
    Rel(leaks, reddit, "Busca", "HTTPS/OAuth")
```

## Nível 3 — Componente (Módulo de Preço)

```mermaid
C4Component
    title Componentes do Módulo de Preço
    Container(gateway, "Camada Discord", "discordgo")
    System_Ext(cheapshark, "CheapShark API")
    ContainerDb(cache, "Cache TTL", "in-memory")

    Container_Boundary(price, "Módulo de Preço") {
        Component(handler, "Handler /preco", "Go", "Valida input, orquestra a busca e monta o embed")
        Component(provider, "PriceProvider", "interface Go", "Contrato de busca de preços")
        Component(client, "CheapShark Client", "Go", "Implementa PriceProvider; HTTP com retry/backoff e tratamento de 429")
    }

    Rel(gateway, handler, "invoca")
    Rel(handler, provider, "usa")
    Rel(provider, client, "implementado por")
    Rel(handler, cache, "consulta/atualiza")
    Rel(client, cheapshark, "GET", "HTTPS")
```
